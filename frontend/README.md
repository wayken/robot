# 简介

Teambeit Robot 是一个基于Electron的AI Agent管理应用程序，旨在为用户提供一个集成的环境来管理和使用各种AI Agent。通过Teambeit Robot，用户可以轻松地创建、配置和运行AI Agent，以满足不同的需求和场景。
